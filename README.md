# E-Commerce Data Warehouse
An end-to-end data warehouse project built on PostgreSQL with a star schema design, incremental ETL, SCD Type 2 customer and product tracking, PySpark transformations, and Airflow orchestration.

## Tech Stack
- PostgreSQL (via Docker)
- Python ETL
- Apache Airflow
- PySpark
- Parquet / Data Lake
